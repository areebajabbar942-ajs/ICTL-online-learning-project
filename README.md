# Online Learning Academy Website
- An Online Learning Academy Website developed as part of our 1st Semester ICT course project. The objective of this project   was to create a clean, interactive, and user-friendly online learning platform with a smooth and logical user experience.

## Technologies Used
- HTML5
- CSS3

## Features
- About Section
- Courses Page
- Team Introduction Page
- Feedback System
- Contact Page
- User Registration & Login Interface
- Enrollment Process with Payment Flow

## Learning Outcomes
Through this project, we gained hands-on experience in:
- HTML page structuring
- CSS styling and layouts
- User interface design
- Front-end development practices
